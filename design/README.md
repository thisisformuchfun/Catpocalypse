# Design Directory

Game design documentation, level designs, and narrative content.

## Structure

- **levels/** - Level layouts, maps, progression
- **narrative/** - Story, dialogue, character bios
- **mechanics/** - Detailed mechanics documentation

## Contents

### Levels

Document each level/area with:
- Layout sketch/map
- Objectives
- Cat spawn points
- Item placements
- Environmental storytelling elements

### Narrative

- Story beats and pacing
- Dialogue scripts
- Character descriptions
- World-building notes

### Mechanics

- Detailed mechanics breakdowns
- Balance spreadsheets
- Item stats
- Cat behavior parameters

## Tools

Recommended tools for design docs:
- **Diagrams**: Draw.io, Excalidraw
- **Maps**: Tiled, paper sketches photographed
- **Dialogue**: Yarn Spinner, Twine, or simple markdown
- **Spreadsheets**: Google Sheets, Excel

## Iteration

Design docs are living documents. Update as you prototype and playtest!
